React E-commerce Product Filter
irish-dance-shop.netlify.app
This is a React-based e-commerce frontend featuring advanced product filtering (category, company, price range, and search).
The project is built using Create React App, React hooks (useState), and custom CSS.
All product data is stored locally on the frontend (no backend).

Features

Live search by product title

Filter by:

Category
Company
Price range


Reusable components (Sidebar, Navigation, Products, Cards)
State management with useState
Custom CSS layout with fixed sidebar
Local product database (no API required)


Tech Stack

React (Create React App)
JavaScript (ES6+)
CSS (Flexbox layout)
React Hooks (useState)
Local data source (db/data.js)


Start the development server
npm start
